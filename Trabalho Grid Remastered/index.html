Trabalho sobre Grid Layout CSS


Grid são linhas de referência para o conteúdo da página, seja em métodos impressos ou digitais. Eles auxiliam para o alinhamento de elementos e de objetos

by: Cristofer Vargas

Grid na Web

Utilizar métodos de alinhamento ou posicionamento de elementos com um grid, auxilia muito na qualidade visual do seu site ou projeto.

Quando ocorre em páginas na web, deve-se dar um grande destaque aos elementos de importância, pois à toda uma hierarquia dos elementos: Dos que chamam mais atenção, trazendo foco ao leitor, como títulos e tópicos de interesse, aos que são descritivos de imagens e também são próprios como texto, para leitura prolongada.

Ao mesmo tempo que o grid define o alinhamento dos textos, ele é responsável por delimitar certas “áreas” do seu layout, como por exemplo, a separação de cabeçalhos, conteúdos principais e seções no corpo da página, como também o rodapé. Comumente prático para determinar um padrão de “estilo” para o decorrer da página e estabelecer um visual agradável.

“Grid é uma malha que divide a tela ​em partes proporcionais e possibilita ​a distribuição do conteúdo de forma​ uniforme, proporcionando equilíbrio visual e estrutural. Construir um Design Responsivo se torna muito mais fácil se ele for apoiado em um Grid Flexível.”

Benefícios do Grid… para o usuário

Facilita a leitura e o escaneamento dos elementos. ​

Melhora a experiência visual.​

Menos esforço cognitivo para interpretar o conteúdo.​

O olho humano percebe a distribuição proporcional dos elementos e associa as informações com maior facilidade.

Benefícios do Grid… para o designer

Rapidez para adicionar, alterar e mover elementos.

Padrões de espaçamento e alinhamento proporcionam uma linha de trabalho uniforme.​

Linguagem visual comum em toda a página.

Mantém a consistência do trabalho caso outro designer mexa no arquivo.

Benefícios do Grid… para o desenvolvimento

É o caminho para que o layout que irá ao ar fique o mais “pixel perfect” possível.​

Atualização do layout de maneira consistente.

Rapidez na construção do front-end do site.

Grid Auxilia a responsividade

Um design responsivo trata-se de um conjunto de técnicas para adaptar a página e seus elementos, para que sejam organizadas em telas menores, ou resoluções diferentes das "comuns" e mais utilizadas.

O design responsivo serve para melhorar a experiência do usuário em diferentes telas, e não “bugar” os elementos do site, deixando-os ainda funcionais e interativos.

Grid Responsivo

Quando a um Layout responsivo, a primeira coisa que tem que saber é que tudo funciona com base em porcentagens(%) ou métricas “flutuantes” que se expandem ou se adaptam na medida necessária que o monitor ou a resolução sofre alterações. O Grid Fluído, como o nome diz, vai sempre dar a impressão de ocupar toda a tela ou se expandir de acordo com a necessidade.

A também métodos como as Media Queries, que possibilitam mudar o estilo ou layout dos objetos a medida que uma certa regra é atingida, como por exemplo, a própria resolução do monitor. Essa regra pode ser aplicada nas Media Queries ou ao grid utilizando métodos de quebra de linha ou quebra de coluna

Qual melhor formato?

O formato 960gs foi bastante usado durante muitos anos, quando a resolução mais popular era 1024x768px. Nos dias de hoje, 13,28% dos monitores utilizam 1024x768px como resolução. A grande maioria utiliza 1366x768px como resolução. Ou seja, utilizar o Grid 960g pode tornar o design limitado e preso a uma grade ultrapassada. O mais comum, vem sendo utilizar uma largura de 1140px, para que no Grid para que se adapte bem às maiores resoluções de monitores.

Números de Colunas

O formato mais utilizado, é de 12 colunas, por causa da flexibilidade proporcionada e por ser capaz de dividir o layout uniformemente. Ou seja, você pode dividir 12 por 2,3,4 e ter uma série de modulações disponíveis (2 x 6 , 3 x 4) e também em várias outras combinações. Se você dividir 12 colunas por 2, vai perceber que consegue ter 6 conjuntos de 2 colunas sem que sobre nenhuma coluna.

Mas o número de colunas pode variar de projeto para projeto. Basicamente, quanto maior o número de colunas, (14 ou 21, por ex.) maior o quantidade de possibilidades para dividir o layout. Geralmente esse tipo de Grid é utilizado quando se tem muito conteúdo editorial e o design é projetado com uma largura de 1140px ou mais.
